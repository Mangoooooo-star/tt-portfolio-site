# 在同一个 UI 页面里新增项目的方法

你的目标不是新增链接入口，也不是新增独立页面。

而是：

- 在 `ui` 子页面里
- 直接新增一个完整的设计项目
- 和现有项目一样，从“项目头图模块”开始
- 一直到下面所有内容模块，完整展示在同一个页面中

## 你以后应该怎么做

1. 打开这个模板文件：
   - `ui-project-template/module-template.html`
2. 复制里面整段：
   - `<section class="ui-project-module"> ... </section>`
3. 把它插入到 `ui/index.html` 里
4. 放在现有项目模块的上方或下方
5. 再替换：
   - 项目标题
   - 副标题
   - 左侧蓝色卡文案
   - 数字指标
   - 每个内容模块的标题
   - 每个内容模块的图片

## 这个模板的意义

它不是“新页面模板”，而是“同一个页面里的项目块模板”。

也就是说：

- 你现在的 `ui/index.html` 以后可以同时展示多个项目
- 每个项目都按同样的模块结构往下排
- 从而保持页面结构和交互一致性

## 最重要的提醒

你当前的 `ui/index.html` 是 UEmo 导出的压缩 HTML，文件结构很不利于手工维护。

所以：

- 这个模板适合作为内容结构参考
- 如果你要长期在同一个 UI 页面里持续增加多个项目
- 最稳的方案是后续把 `ui/index.html` 重构成可维护版本

## 推荐下一步

如果你准备真的开始加第二个 UI 项目，最合适的做法是：

1. 你把新项目的标题、说明和图片准备好
2. 我直接帮你把第二个项目模块真正插进 `ui/index.html`
3. 这样你以后就有一个已经落地的“双项目 UI 页面”样板
