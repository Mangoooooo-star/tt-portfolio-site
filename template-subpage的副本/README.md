# Subpage Template

## How to use

1. Copy the whole `template-subpage` folder
2. Rename it with a lowercase English name
   - Example: `motion`
   - Example: `campaign`
3. Edit `index.html`
4. Replace the title, text, and image paths
5. Add a new homepage entry that points to `./your-folder-name/`
6. Commit and push with GitHub Desktop

## Important rules

- Folder names should use English lowercase only
- Avoid spaces and Chinese characters in folder names
- Compress large images before putting them into the repo
- Keep the main cover image under about 300 KB when possible

## Suggested workflow

1. Duplicate template
2. Replace copy and images
3. Check locally in browser
4. Commit in GitHub Desktop
5. Push to GitHub Pages
